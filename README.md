# Verbum — a 3D Bible Explorer

Scripture renders as an open 3D book — curved pages, typeset text, rubric-red
verse numbers — that you can page through from Genesis to Revelation. Click any
word and the passages bound to its verse fly in as parchment cards, tied back
to the word by golden threads. Click a card and the book riffles its pages to
that passage; the ↶ / ↷ history buttons return you to your exact spot,
selection included.

- Pages are laid out **per book** with inline chapter headings; folio numbers
  run continuously through each book.
- Page turns are a hinged double-faced mesh (TSL `frontFacing` texture select)
  and cross chapter and book boundaries.
- Word picking is raycast → page UV → typeset token rects; highlights sit on
  the curved page surface.

![concept](docs/screenshot.png)

## Stack

- **Three.js WebGPURenderer + TSL** (`three/webgpu`, `three/tsl`) — background
  gradient, flickering dust motes, animated thread pulses are all node
  materials. Falls back to WebGL2 automatically where WebGPU is unavailable.
- **Plain DOM reading panel** — real text for crisp typography, accessibility,
  and per-word click targets; the 3D layer lives behind it through a
  translucent backdrop.
- **Vite**, no framework.

## Data

Generated by `npm run data` (gitignored, ~9 MB):

- **Text**: Douay-Rheims (public domain, full 73-book Catholic canon) from
  [getbible.net](https://getbible.net).
- **Cross-references**: ~340,000 weighted verse connections from
  [OpenBible.info](https://www.openbible.info/labs/cross-references/) (CC-BY),
  derived from the Treasury of Scripture Knowledge.

The pipeline remaps the dataset's Hebrew (KJV) Psalm numbering to this
edition's Septuagint/Vulgate numbering, validates every reference against the
actual text, drops downvoted links, and keeps the top 14 references per verse.
Known limitation: a handful of non-Psalm versification differences (e.g.
Joel 2/3, Malachi 3/4 chapter splits) are not yet remapped.

## Run

```sh
npm install
npm run data   # one-time: download + process text and cross-references
npm run dev
```

Deep links: `/#Bookname.Chapter` (OSIS id), plus `?sel=verse[.wordIndex]` to
open a verse's threads on load — e.g. `/?sel=1#John.1`.

`node scripts/snap.mjs <url> <out.png> [waitMs] [x,y[,afterMs]]` screenshots
the app in headless Chrome over CDP (dev helper).
